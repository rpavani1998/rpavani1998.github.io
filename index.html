PAV·AI is a real RAG

A full retrieval-augmented generation pipeline over this portfolio. The site you're on becomes the knowledge base.

  1. Retrieve. Your question is embedded and cosine-matched against ~28 chunks ingested from this site. The closest ones above a relevance threshold win.
  2. Augment + generate. Those chunks ground Llama-3.1-8b, which answers only from them and refuses when nothing's relevant.
  3. Guarded. Prompt-injection resistant, input/output capped, CORS-locked, no API keys in the browser. All on a free Cloudflare Worker.

Tested with an eval harness (retrieval, groundedness, refusal, injection). A deliberate reference build: my résumé fits in context, but I built the full pipeline to show it done right.
